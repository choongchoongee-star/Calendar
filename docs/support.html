자주 묻는 질문 (FAQ)

Q. 로그인 없이 사용할 수 있나요?

네. 게스트 모드로 로그인 없이 모든 기능을 사용할 수 있습니다. 단, 게스트 모드에서는 데이터가 기기에만 저장되며 기기 간 동기화나 공유 기능은 사용할 수 없습니다.

Q. 일정을 다른 기기와 동기화하려면 어떻게 하나요?

Apple 또는 Google 계정으로 로그인하시면 동일 계정을 사용하는 모든 기기에서 자동으로 일정이 동기화됩니다.

Q. 위젯이 업데이트되지 않아요.

  • 홈 화면에서 위젯을 길게 눌러 "위젯 편집"에서 연결된 계정 확인
  • 앱을 한 번 열어 최신 데이터를 동기화
  • iOS는 위젯 새로고침 주기를 자체적으로 관리하므로 즉시 반영되지 않을 수 있습니다 (보통 수 분 이내)
  • 계속 문제가 있다면 위젯을 제거 후 다시 추가해 주세요

Q. 알림이 오지 않아요.

  • iOS 설정 → 채우다 → 알림 허용 확인
  • 방해 금지 모드 / 집중 모드 해제 확인
  • 일정에 알림 시간이 설정되어 있는지 확인

Q. 공휴일이 표시되지 않아요.

현재 버전은 2025년부터 2030년까지의 대한민국 공휴일(대체공휴일 포함)을 기본 제공합니다. 2031년 이후 공휴일은 업데이트를 통해 추가될 예정입니다.

Q. 계정을 탈퇴하고 싶어요.

앱 내 설정 화면에서 '계정 탈퇴' 버튼을 누르면 서버에 저장된 모든 데이터가 즉시 삭제됩니다. 이 작업은 되돌릴 수 없으니 신중히 선택해 주세요.

Q. 데이터를 백업하거나 내보낼 수 있나요?

로그인 상태로 사용하시면 Firebase 서버에 자동 백업되어 앱을 재설치해도 데이터가 유지됩니다. 별도의 파일 내보내기 기능은 향후 업데이트에서 추가될 예정입니다.
